- **설명**: AI 기반 사이버 수사 시스템

## 🚀 빠른 시작

### 설치
```bash
# 루트 디렉토리에서
pip install -r requirements.txt
```

### 실행
```bash
# 이 프로젝트 실행
python run_cyber_detective.py

# 또는 루트에서 직접 실행
cd ../..
python cyber_detective_ai.py
```
