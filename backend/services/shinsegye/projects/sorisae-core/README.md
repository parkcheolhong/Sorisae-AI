## 📋 프로젝트 정보

- **카테고리**: Core Systems
- **설명**: Two-Cycle Brain AI 핵심 시스템
- **포트**: 5050

## 🚀 빠른 시작

### 설치
```bash
# 루트 디렉토리에서
pip install -r requirements.txt
```

### 실행
```bash
# 이 프로젝트 실행
python run_sorisae_core.py

# 또는 루트에서 직접 실행
cd ../..
python run_all_shinsegye.py
```
