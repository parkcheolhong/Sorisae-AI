- 🏨 **숙소** (2): 예약/요금
- 🚖 **교통** (2): 택시/역

- ☀️ **날씨** (2): 내일/비

## 🌐 2️⃣ 하이브리드 통역 시스템 (`hybrid_interpreter_system.py`)

**🎯 목적**: 비지니스 & 엔터프라이즈급 고가용성
**📱 사용처**: 국제 회의, 원격 협업, 중요 통역

### 핵심 기능

#### 🔄 3-Way 자동 연결 전환

```python
1. 온라인 (기본)
   → 고품질 API 번역
   → 실시간 반영

2. 위성 (백업)
   → 인터넷 불가 시 자동 전환
   → 통신 지연 허용

3. 오프라인 (최후 수단)
   → 로컬 패턴 매칭
   → 기본 문구만
```

#### 📊 Request 추적

```python
- Request ID: 고유 식별자
- Timestamp: 정확한 시간 기록
- Response Time: 성능 모니터링
- Confidence Score: 번역 신뢰도
```

#### 🎤 음성 통역

```python
음성 인식 → 번역 → 음성 합성
(완전한 실시간 회의 통역)
```

### 지원 언어 (13개)
ko, en, ja, zh, es, fr, de, ru, ar, vi, th, id, sorisae

## 💬 4️⃣ 자유 대화 & 통역 시스템 (`hybrid_conversation_translator.py`)

**🎯 목적**: 실시간 자유로운 대화 + 자동 통역
**📱 사용처**: 국제 회의, 멀티팀 협업, 라이브 방송

### 핵심 기능

#### 🗣️ 완전한 자유 대화

```python
- 사용자가 자유롭게 말함
- 시스템이 자동으로 통역
- 13개 언어 모두 지원
```

#### 🛰️ 하이브리드 연결 관리

```python
연결 품질 모니터링
    ↓
자동 전환 (모바일→지상파→위성)
    ↓
대화 지속 (무중단)
```

#### 📱 멀티 연결 지원

```python
- 지상파 (Terrestrial): 일반 인터넷
- 모바일 (Mobile): 4G/5G
- 위성 (Satellite): 인터넷 없는 지역
```

#### 🔐 대화 이력 관리

```python
ConversationMessage 객체:
- timestamp: 시간
- user_text: 사용자 발화
- bot_response: 번역 응답
- language: 언어
- connection_type: 연결 종류
- confidence: 신뢰도
```

#### ⏸️ 완벽한 종료 처리

```python
SHUTDOWN_REQUESTED 플래그로 안전한 종료
- 대화 이력 저장
- 캐시 정리
- 연결 해제
```

### 지원 언어 (13개 예상)
ko, en, ja, zh, es, fr, de, ru, ar, vi, th, id, ...

**상태**: ⏳ **구현됨 (검증 대기)**
