## 3. 장애 대응표

| 장애 유형 | 원인 후보 | 대응 |
|---|---|---|
| request 미매핑 | endpoint/client/secret 미입력 | 관리자 대시보드 운영값 저장 후 상태 카드 재검증 |
| complete 미매핑 | callback URL 미등록 | 공급사 callback 등록 후 저장값 재확인 |
| 401/403 | 운영 키 불일치 | client id/secret 재발급 확인 |
| 404 callback | callback 경로 오입력 | 공급사 등록 URL과 백엔드 경로 일치 확인 |
| 과금 과다 | 실패/재시도 과금 포함 | 계약서에서 실패/재시도 과금 조건 재확인 |
