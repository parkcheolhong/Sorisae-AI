# WorldLinco 통역 통화 베타 — 이용 안내 (v1.0)

> **대상:** 개인 Android APK · 한국에 사는 내국인·외국인  
> **버전:** build **66** (`1.0.41`) · **상태:** 베타 (무료)  
> **마켓:** [개발분석114 마켓플레이스 WorldLinco](/marketplace/worldlinco)

---

## WorldLinco란?

**친구와 전화하면, 말하는 대로 상대방 언어로 들려주는** 실시간 통역 통화 앱입니다.

한국은 다문화 사회로 변하고 있습니다. 가족, 친구, 연인, 이웃과 **언어가 달라도 전화로 대화**할 수 있도록 만들었습니다.

## 설치 방법

1. [마켓플레이스 WorldLinco 페이지](/marketplace/worldlinco)에서 **로그인** (회원가입 무료)
2. **APK 다운로드** → Android 기기에 설치  
   - 「출처를 알 수 없는 앱」 설치 허용 필요
3. 앱 실행 → 가입·로그인 → **친구 추가** → **통역 통화** 시작

**패키지:** `com.parkcheolhong.worldlinco`  
**최소:** Android 7+ (API 24)

---

## 통역 통화 사용 팁

1. **WiFi**에 두 기기 모두 연결
2. 통화 중 **마이크·스피커** 권한 허용
3. 한 문장 말하고 **잠깐 멈추기** — 상대방 TTS가 끝난 뒤 다음 말하기 (동시에 말하면 겹칠 수 있음)
4. Tab 등 **큰 스피커** 사용 시 이어폰 권장 — 스피커 소리가 마이크에 들어가면 인식이 꼬일 수 있음

---

## 알려진 이슈 (build 65)

- **Tab 스피커 + TTS 피드백:** 반복 인식 버그 → build 65에서 `repetition_hallucination` 차단 적용. 재발 시 피드백 부탁드립니다.
- **자동 E2E:** 스크립트 기준 accept·connected가 가끔 flaky — **수동 통화**가 더 안정적일 수 있습니다.
- **서버 필요:** 통역·통화는 **인터넷·백엔드 연결** 필요 (완전 오프라인 아님)
